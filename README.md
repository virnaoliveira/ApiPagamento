# Desafio ApiPagamento

Essa api foi desenvolvida para lojas virtuais, que avalia se o valor mínimo de compra(R$100) foi atingido. Se sim, aprova a compra, se não, reprova.

### 📋 Pré-requisitos

* Visual Studio
* Sql Server

## 🛠️ Construído com

* C#
* REST Api
* EntityFramwork Core
* .Net Core
